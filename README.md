# 🎬 Netflix PySpark Analysis

A data analysis project using **PySpark** to explore and analyze the Netflix dataset. This project demonstrates how PySpark can efficiently process large datasets and uncover meaningful insights through data cleaning, transformation, and analysis.

## 📌 Features

- Data loading using PySpark
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Trend analysis and visual insights
- Efficient handling of large datasets

## 🛠️ Technologies Used

- Python
- PySpark
- Jupyter Notebook
- Pandas
- Matplotlib

## 📊 Dataset

The project uses a Netflix dataset containing information such as:
- Title
- Type (Movie/TV Show)
- Director
- Cast
- Country
- Release Year
- Rating
- Duration
- Genre
